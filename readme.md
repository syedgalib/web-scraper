# Web Scraper

## Requirements
1. [Node JS](https://nodejs.org/en/download/package-manager) ( v22.7.0 or later )

## Instalation
1. Clone Github Repository
   ```sh
   git clone https://github.com/syedgalib/web-scraper.git
   ```

2. Install node dependencies
   ```sh
   yarn
   ```
